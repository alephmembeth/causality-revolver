# Answers at Gunpoint

This is a repository for the paper [»Answers at Gunpoint«](https://doi.org/10.1017/psa.2021.21) by [Jan Romann](https://github.com/JKRhb) and me, published in _Philosophy of Science_.

- `/analysis` contains data and a script for analysis with R (version 4.2.2).
- `/preprint` contains the _Philosophy of Science_ preprint.[^1]
- `/study` contains files from the study created with _LimeSurvey_ (version 4.1.18+200416), using the theme `extends_vanilla`.

[^1]: Bauer, Alexander Max, and Jan Romann (2022): »Answers at Gunpoint. On Livengood and Sytsma’s Revolver Case«, _Philosophy of Science_ 89 (1), 180–192. ([Link](https://doi.org/10.1017/psa.2021.21))
